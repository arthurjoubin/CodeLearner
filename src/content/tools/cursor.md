## Overview

Cursor is an AI-powered IDE built as a fork of VS Code, offering deep AI integration at every level of the development workflow.

## Key Capabilities

- **AI Autocomplete**: Intelligent code completion that understands context
- **Natural Language Editing**: Edit code by describing what you want
- **Code Explanation**: Get instant explanations of any code block
- **Bug Detection**: AI-powered error detection and fixing suggestions

## Use Cases

Cursor excels in:
- Daily development workflows
- Refactoring large codebases
- Learning new codebases
- Rapid prototyping
