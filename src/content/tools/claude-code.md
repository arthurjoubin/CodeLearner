## Overview

Claude Code is Anthropic's autonomous coding agent that goes beyond suggestions to actually execute code, manage files, and interact with external systems.

## Key Capabilities

- **Autonomous Execution**: Run shell commands and manage files
- **Multi-file Reasoning**: Understand and modify code across entire projects
- **Git Integration**: Commit, branch, and manage version control
- **Test Execution**: Run tests and fix failures automatically

## Use Cases

Claude Code excels in:
- Complex refactoring tasks
- Multi-file changes
- Debugging and error fixing
- Project initialization
